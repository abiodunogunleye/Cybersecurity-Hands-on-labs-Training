## Objective
Learn the basics of Bash scripting to automate common tasks during investigations.

## Commands Practiced
nano script.sh  
chmod +x script.sh  
./script.sh  
Variables: name="test"  
Loops and conditions: if, for, while  
echo for output  
Basic automation tasks

## Key Takeaways
• Scripting supports automation for SOC workflows.  
• Variables and conditions help create flexible tasks.  
• Execution rights must be granted for scripts.  
• Bash scripts improve speed during log analysis.

## SOC Application
SOC analysts use scripts to collect logs.  
They automate repeated commands.  
They combine scripting with grep for quick searches.
